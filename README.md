## 🚀 How to Use

1. Rename your sister's photo to **photo.jpg**.
2. Add your birthday song as **music.mp3**.
3. Upload all files to your GitHub repository.
4. Commit the changes.
5. Enable GitHub Pages.

---

## 🌐 GitHub Pages

Go to:

**Settings → Pages**

Select:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

Click **Save**.

After a few minutes your website will be live.
